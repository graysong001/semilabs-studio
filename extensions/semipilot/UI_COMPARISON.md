## 🔍 GitHub Copilot 风格特征

### ✅ 已实现特征

1. **顶部标题栏布局**
   - 左侧: 图标 + 标题
   - 右侧: 操作按钮（水平排列）

2. **黑白 SVG 图标**
   - 所有图标使用 `fill="currentColor"`
   - 跟随 VS Code 主题颜色
   - 无彩色元素

3. **透明按钮设计**
   - 默认透明背景
   - Hover 时显示 `toolbar-hoverBackground`
   - 使用不透明度表达状态

4. **箭头发送按钮**
   - 简洁的箭头图标 (→)
   - 与 GitHub Copilot 完全一致

5. **统一输入框**
   - 单列垂直布局
   - 工具栏在底部
   - 干净整洁
